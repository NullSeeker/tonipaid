ToniPaid 

ToniPaid is a lightweight, web-based personal finance and wealth management application. It helps users track their cash and cashless balances, log transactions, and intelligently scan receipts using AI.

Features

 
- Smart Dashboard: Visual progress bars for monthly budgets and separate tracking for Wallet (Cash) and Bank (Cashless) accounts.
- AI Receipt Scanner: Uses Optical Character Recognition (OCR) via Tesseract.js to automatically read receipt images and extract the total amount spent.
- Transaction History: Keep a detailed log of all money coming in and going out.
- Responsive Design:  A sleek, mobile-friendly interface built with custom CSS.




# How to Run Locally

If you want to download the code and run it yourself:
1. Clone this repository.
2. Open the project folder in VS Code.
3. Install the "Live Server" extension.
4. Right-click `index.html` and select "Open with Live Server".
